## 功能特点
- 支持省份面板生成
- 支持城市面板生成
- 支持自定义面板生成
- 自动更新功能
- 多种导出格式（Excel、PDF）
- 数据可视化
- 数据分析
- 批量处理
- 多语言支持
- 主题切换
- 错误处理
- 日志系统

## 系统要求
- Windows 7 及以上版本
- macOS 10.13 及以上版本
